# PsyReaSFX 0.6.17 Stable RC6 用户使用说明书

## 1. 产品定位

PsyReaSFX 是运行在 REAPER 内部的音效资产浏览、搜索、试听、波形选区、Region、元数据、集合管理与素材插入工具。界面采用高密度资产表格与模块化工作区，并提供 Forge Compact 与 Aether Standard 两套原创工作流预设。

本手册只说明当前稳定可用的 0.6 功能。版本变化记录见《更新日志》。

## 2. 运行要求

### 必需

- REAPER 7.x
- ReaImGui 0.10 或更新版本

### 强烈建议

- SWS Extension

SWS 用于点击波形精确定位、选区试听、Pitch/Rate/Gain、左右声道监听、拖入 REAPER 编排区以及部分 Preview 参数控制。没有 SWS 时，基础浏览、搜索和索引仍可使用，但高级试听与拖放会受限。

## 3. 安装与升级

1. 解压发布包。
2. 打开 REAPER 的 Action List。
3. 点击 `ReaScript: Load...`。
4. 加载 `PsyReaSFX_v0_6_17_Stable_RC6.lua`。
5. 将原快捷键改绑到新脚本。
6. 停止旧版 PsyReaSFX，避免多个版本同时运行。

数据默认保存在：

```text
<REAPER Resource Path>/Scripts/PsyReaSFX/
```

升级时会继续读取已有库、索引、集合、Region、响度和波形缓存。0.6.10 会继续读取旧索引，并在保存时补充 Artwork 路径等新字段；无需手动迁移。

## 4. 主界面

主界面分为五个区域：

- 顶部：搜索、自动试听、扫描、设置、面板开关。
- 左侧：音效库、播放列表、项目素材箱、保存搜索、状态筛选。
- 中央：固定表头和音频结果列表。
- 右侧：非破坏性元数据查看与编辑。
- 底部：高精度波形、时间标尺、响度、Region、试听与插入控制。

按 `F9`、`F10`、`F11` 可分别切换左栏、右栏和专注模式。


## 4.1 外观与设置中心

0.6.11 的设置窗口采用与主界面一致的深色双栏设置中心：左侧为页面导航，右侧为可滚动设置内容，底部保留统一的保存按钮。

设置页分为：

- **常规**：语言、左右面板、插入命名和淡化；
- **外观**：界面预设、密度、层级、控制台、Artwork 和状态颜色；
- **波形**：波形精度、预缓存、缩放、瞬态、响度；
- **维护**：缓存、重建数据库和恢复出厂；
- **关于**：产品版本、运行环境、数据目录、诊断信息和支持信息。

### 界面预设

```text
设置 → 外观 → 界面预设
```

- **Forge 紧凑**：单行高密度结果列表，默认显示 Waveform、Filename、Keywords/Description、Artwork、Duration；
- **Aether 标准**：保留分层面板与较舒展的列表；
- **检查/审阅**：强化状态、元数据和对比显示。

预设会一次性调整密度、层级和字段组合，随后仍可单独修改列、颜色和控制台。

### 密度与层级

- **舒适 / 标准 / 紧凑**会实际改变行高、表头、图标、间距和圆角；
- **扁平 / 分层 / 高对比**控制面板色差、网格和文字对比；
- **轻量工具条 / 专注 / 极简**控制底部试听工具条的信息量。

### 结果列表横向导航

当可见字段宽度超过中央工作区时，列表底部不再常驻一条粗大的系统滚动条。将鼠标移入结果区后，固定表头下沿会短暂显示一条细型浮动导航；可直接拖动，也可使用 `Shift + 鼠标滚轮` 横向移动。固定表头会与内容同步。

### 独立声道大波形

```text
设置 → 波形 → 独立显示各声道
```

- 单声道显示 `M`；
- 立体声分为 `L`、`R` 两条波形；
- 多声道分为 `CH 1` 至 `CH 8`；
- 选区、Region、时间标尺和播放指针贯穿全部声道，便于比较声道间的内容与同步关系。

独立声道使用新的 `RWF3` 高精度缓存。列表缩略图仍兼容既有 `RWF2` 缓存，不会因为升级而重建整个缩略图库。高精度预缓存会生成可直接复用的独立声道数据。

### Forge 紧凑列表

紧凑模式改为单行垂直居中文本，不再在有限行高中强行绘制第二行分类文字，因此不会出现下半行被裁切的问题。可通过以下两处应用：

```text
设置 → 外观 → Forge 紧凑
表头右键 → Forge 紧凑预设
```

### 设置导航

左侧导航使用自绘双行卡片：第一行为页面名称，第二行为功能摘要。导航宽度和行高固定，不再依赖 `Selectable` 自动裁切，因此中文与英文标签都能完整显示。

### 色盘与即时预览

所有可修改颜色统一使用 ReaImGui 色盘：

```text
设置 → 外观 → 界面主题 / 波形配色
```

- 点击色块打开色盘；
- 选择颜色后立即应用到主界面；
- 色块旁显示当前 `#RRGGBB` 值，仅作为识别信息；
- 每个颜色项都有独立恢复按钮；
- 不再需要手动查找或输入颜色代码。

可调整项目包括强调色、普通/选中/已播放/已标记波形、已播放文字、选区、播放指针和 Region。


### 色块无法打开时

0.6.12 RC1 已移除覆盖色块的整行 `InvisibleButton`。颜色行现在使用非交互占位区域，点击事件会直接交给色块。

1. 点击颜色项右侧的小色块；
2. 色盘打开后直接选择颜色；
3. 颜色即时应用；
4. 点击“恢复”可还原该项默认值。

### About 页面

```text
设置 → 关于
```

页面显示：

- PsyReaSFX 版本与作者；
- REAPER、操作系统、ReaImGui 与 SWS 检测状态；
- 当前试听后端；
- 数据目录；
- 打开数据目录、打开文档目录；
- 复制诊断信息。

许可、官方网站与支持联系方式目前保留为待配置项。


## 5. 音效库

### 添加库

点击左侧 `+ 添加音效库`，输入素材根目录。PsyReaSFX 会递归扫描子目录，并按阶段建立元数据和列表波形。

### 导入进度

新库导入会显示：

- 当前阶段和文件；
- 完成数、失败数和总数；
- 波形准备进度；
- 取消按钮。

### 管理库

`管理音效库` 支持打开目录、重建索引和从 PsyReaSFX 删除库。删除库不会删除硬盘音频文件。

## 6. 搜索

### 普通搜索

```text
cinematic whoosh
metal impact
magic ui
```

### 字段搜索

```text
category:impact
subcategory:metal
catid:WPN
library:boom
description:heavy
keywords:magic
channels:2
status:candidate
marked:true
played:true
```

### 排除词

```text
whoosh -long
impact -debris
```

多个条件可组合使用。

## 7. 表头与字段

表头固定置顶。拖动列分隔线可修改列宽，双击恢复默认宽度。右键表头可显示或隐藏 Waveform、Filename、Status、Description、Category、Duration、Format、Library、Path 等字段。

当字段总宽度超过中央工作区时，结果列表底部会显示横向滚动条。滚动时固定表头与行内容同步移动，最右侧字段和快捷键信息不会再被截在窗口外。用户设置的列宽会被保留，不会为了适应当前窗口而自动压扁全部字段。

隐藏字段不会参与行内绘制，有利于大型结果列表的性能。

### Artwork 列

在表头右键菜单勾选 `Artwork` 后，列表会显示方形封面缩略图。Forge 紧凑预设会默认启用该列。

Artwork 查找顺序：

1. 当前素材所在文件夹；
2. 逐级向上查找到音效库根目录，最多七层；
3. 优先识别 `artwork`、`cover`、`folder`、`front`、`album`、`thumbnail` 等 PNG/JPEG 文件；
4. 没有匹配名称时使用文件夹中第一张可用 PNG/JPEG；
5. 可在右侧元数据面板手动选择封面。

封面使用低优先级队列按需查找，不会在启动时扫描整库图片。图片内存缓存有数量上限，隐藏 Artwork 列时不会为不可见行加载缩略图。


## 8. 选择、多选与标记

| 操作 | 功能 |
|---|---|
| 单击 | 单选 |
| Ctrl+单击 | 追加或取消单项 |
| Shift+单击 | 连续范围选择 |
| Ctrl+A | 全选当前结果 |
| F | 收藏或取消收藏 |
| M | 标记或取消标记 |

“收藏”和“标记”用途不同：收藏用于长期集合；标记用于当前筛选、视觉检查和临时处理状态。可使用 `marked:true` 搜索标记素材。

右键素材可批量收藏、标记、设置工作流状态、加入集合或分轨插入。

## 9. 状态颜色与本次已播放高亮

打开：

```text
设置 → 外观 → 颜色与状态
```

可设置：

- 普通、选中、已播放、已标记波形；
- 已播放文字颜色；
- 选区、播放指针和 Region 颜色。

### Forge 式已播放文字

素材在本次启动中真正开始试听后，Filename、Keywords/Description、Duration、Library 等文字列会改为可自定义的已播放颜色，默认接近 Forge 的暖黄色。选中行仍保留蓝色背景，但已播放文字不会被蓝色选中状态覆盖。

这套颜色属于**当前启动会话**：

- 重新打开脚本后自动恢复普通文字颜色；
- 不会删除 `history_v1.tsv` 中的长期试听历史；
- `played:true` 搜索仍表示历史上试听过的素材；
- 主工具栏的“清除本次已播放高亮”图标可立即重置；
- 也可通过 `视图 → 清除本次已播放高亮` 或设置页执行。

可单独关闭“已播放文字高亮”。已播放波形高亮默认关闭，避免与标记、选中状态产生过多颜色竞争。

列表波形显示优先级仍为：

```text
选中 > 已标记 > 已播放波形（启用时） > 普通
```


## 10. 列表波形

列表波形默认 256 点，可切换为 512 点。点击小波形任意位置，会从对应时间开始试听。当前试听素材显示播放指针。

高精度整库预缓存支持 2048 或 4096 点，按素材逐个写入磁盘，不会把整库高精度波形长期留在内存。

## 11. 大波形

### 导航

- 鼠标滚轮：以鼠标位置为中心缩放；
- Shift+滚轮：水平平移；
- 中键拖动：水平平移；
- 双击：重置缩放；
- 右键拖动：擦播。

### 选区

左键拖动建立选区。可只试听选区、自动循环选区，或通过选区顶部拖拽区域直接拖入 REAPER。

### 时间与响度

顶部信息栏可显示素材名称、时长、声道、采样率、音效库以及可选的 LUFS-I、LUFS-M max、LUFS-S max、True Peak。响度结果按需分析并保存到 `loudness_v1.tsv`。

## 12. Region 与瞬态检测

一条长素材可保存多个非破坏性 Region。Region 数据保存在 `regions_v1.tsv`，不会修改源 WAV。

瞬态检测可设置：

- 阈值 dBFS；
- 包络平滑时间；
- 最小瞬态间隔；
- Region 前置和后置时间；
- 单次最大 Region 数；
- 是否替换已有建议。

自动检测 Region 标记为 `[T]`，手动 Region 标记为 `[M]`。支持撤销上次检测、清除全部瞬态建议或单独删除某个 Region。

## 13. 试听

支持：

- Pitch：-24 至 +24 半音；
- Rate：0.25x 至 4x；
- Gain：-36 dB 至 +18 dB；
- Preserve Pitch；
- Loop；
- Reverse；
- 原始、左、右和 Mono 监听；
- 估算响度匹配；
- Pitch 和 Rate 预设。

估算响度匹配用于快速比较，不等同于完整 LUFS 标准测量，只影响试听。

## 14. 插入与拖放

支持：

- 插入当前轨道；
- 插入新轨道；
- 按 BWF 时间戳插入；
- 只插入当前波形选区；
- 多选素材纵向分轨插入；
- 从列表或选区拖入 REAPER 编排区。

当前拖放是 PsyReaSFX 到 REAPER 的内部工作流，不是 Windows 通用文件拖放。

## 15. 集合与工作流状态

### 播放列表和项目素材箱

可以建立多个集合，批量加入或移除素材。删除集合不会删除文件。

### 保存搜索

保存搜索会记录关键词、库筛选、集合、工作流状态和排序条件。

### 工作流状态

素材可设置为：未标记、候选、已采用、已排除。该状态与 0.6.8 新增的独立 `marked` 标志不同。

## 16. 元数据与固定封面

右侧面板可非破坏性编辑 Description、Keywords、Category、SubCategory、CatID、Library 和 Artwork 路径。多选时必须先勾选需要批量写入的字段。

单选素材且启用 Artwork 时，右侧面板顶部会显示完整封面。默认开启“封面固定”：

- 封面、素材名和封面操作保持在面板顶部；
- 下方元数据与文件信息在独立区域中滚动；
- 滚动长元数据时封面位置不会随内容离开；
- 可手动选择封面、重新自动查找或明确清除封面；
- 关闭固定后，封面会与元数据一起滚动。

当前版本只修改 PsyReaSFX 数据库，不直接写回 WAV/BWF/iXML，也不提取音频文件内部嵌入式图片。Artwork 使用外部 PNG/JPEG 文件。


## 17. 界面与性能

### 界面

支持三个完整界面预设、三档密度、三种层级风格、可调列宽、固定表头、折叠面板和响应式底部控制台。

### 0.6.10 性能处理

- 本次已播放状态使用路径哈希表，查询为常数时间；
- Artwork 使用文件夹负结果缓存，未找到图片的文件夹不会在每帧重复扫描；
- Artwork 查找为低优先级单任务队列，鼠标交互、扫描和导入时主动让步；
- 图片对象通过 ReaImGui Attach 持久化，并限制内存缓存数量；
- 只为当前可见 Artwork 行和选中素材请求图片；
- 紧凑行只绘制一行主文本，减少裁切和重复绘制；
- 设置中心仅在打开时绘制，不加入主浏览循环的数据库工作量。

### 通用性能原则

- 只绘制当前可见行；
- 隐藏字段不绘制；
- 扫描、元数据、响度和波形任务分帧执行；
- 鼠标交互时后台重任务让步；
- 高精度预缓存只写磁盘，不整库常驻内存；
- Region、集合、历史和响度使用独立轻量文件。

## 20. 维护与故障排查

设置中的维护页支持清空波形缓存、重置界面、保留库路径重建数据库和恢复出厂。

常见问题：

- 波形首次显示较慢：等待可见行建立缓存，或执行高精度预缓存。
- 点击波形无法精确定位：安装或更新 SWS。
- 拖放失败：确认释放位置在 REAPER 编排区。
- 中文乱码：通常来自源文件元数据编码，而不是 PsyReaSFX 数据库。
- 大库卡顿：关闭不需要的列，使用专注模式，避免同时运行多个版本。

## 21. 当前阶段
