# PsyReaSFX 0.7.8 Beta 11 用户使用说明书（包版本 0.7.8-beta.11）

## 1. 产品定位

PsyReaSFX 是运行在 REAPER 内部的音效资产浏览、搜索、试听、波形选区、Region、元数据、集合管理与素材插入工具。界面采用一套统一的高密度、自适应工作区，减少重复预设带来的维护和排版差异。

本手册说明当前 0.7 Beta 功能。版本变化记录见《更新日志》。

## 2. 运行要求

### 必需

- REAPER 7.x
- ReaImGui 0.10 或更新版本

### 强烈建议

- SWS Extension

SWS 用于点击波形精确定位、选区试听、Pitch/Rate/Gain、左右声道监听、拖入 REAPER 编排区以及部分 Preview 参数控制。没有 SWS 时，基础浏览、搜索和索引仍可使用，但高级试听与拖放会受限。

## 3. 安装与升级

### ReaPack 安装（推荐）

在 `Extensions → ReaPack → Import repositories...` 中导入：

```text
https://github.com/Psysia/PsyReaSFX/raw/main/index.xml
```

同步仓库后搜索并安装 `PsyReaSFX`。以后只需在 ReaPack 中同步并更新，无需重新下载 ZIP 或重新绑定动作。

### 手动安装

1. 解压发布包。
2. 打开 REAPER 的 Action List。
3. 点击 `ReaScript: Load...`。
4. 加载 `PsyReaSFX_v0_7_8_Beta_11.lua`。
5. 将原快捷键改绑到新脚本。
6. 停止旧版 PsyReaSFX，避免多个版本同时运行。

数据默认保存在：

```text
<REAPER Resource Path>/Scripts/PsyReaSFX/
```

升级时会继续读取已有库、索引、集合、Region、响度和波形缓存。新版会继续读取旧索引并在保存时补充新增字段，无需手动迁移。

## 4. 主界面

主界面分为五个区域：

- 品牌工具栏：应用图标与字标、搜索、面板开关、自动试听、扫描、帮助和设置。
- 左侧：音效库、播放列表、项目素材箱、保存搜索、状态筛选。
- 中央：固定表头和音频结果列表。
- 右侧：非破坏性元数据查看与编辑。
- 底部：高精度波形、时间标尺、响度、Region、试听与插入控制。

按 `F9`、`F10`、`F11` 可分别切换左栏、右栏和专注模式。

点击设置按钮旁的问号图标，可打开跟随当前语言的内置快捷参考。窗口按工作区、结果列表、试听、整理和 Transfer 分组。

Beta 11 已移除额外的应用内菜单条。音效库操作保留在左侧导航，扫描可使用刷新图标或 `Ctrl+R`，Watch Folder 位于**设置 > 常规 > 后台与浏览**。


## 4.1 外观与设置中心

设置窗口采用与主界面一致的深色双栏设置中心：左侧为页面导航，右侧为可滚动设置内容，底部保留统一的保存按钮。

在**外观 > 外观模式**中：

- **黑暗模式**是默认的中性黑色工作区，使用蓝色选中状态、中性波形与绿色播放指针。
- **传统模式**使用更深的藏蓝框架和 Electric Cyan 交互色。
- **框架底色**会打开色盘，并自动派生面板、表头、悬停、边框、弱化文字和波形背景层级。
- **强调色**可以独立自定义。

选择任一预设时，会重新应用完整的框架与波形配色。

设置页分为：

- **常规**：语言、左右面板、插入命名和淡化；
- **外观**：统一布局、下方面板高度、Artwork、主题和状态颜色；
- **波形**：波形精度、预缓存、缩放、瞬态、响度；
- **维护**：缓存、重建数据库和恢复出厂；
- **关于**：产品版本、运行环境、数据目录、诊断信息和支持信息。

### 统一界面

```text
设置 → 外观 → 统一界面
```

当前只维护一套紧凑、扁平、自适应布局。列字段和列宽仍可通过固定表头右键及拖动分隔线调整；左右面板和专注模式仍可独立切换。“恢复统一界面”可恢复默认字段组合，不会删除素材库或数据库。

### 结果列表横向导航

当可见字段宽度超过中央工作区时，不显示任何横向滚动条。将鼠标放在固定表头或结果行上，使用 `Shift + 鼠标滚轮` 横向移动。滚动位置会限制在有效范围内，可完整到达第一列和最右字段；固定表头会与内容同步。

### 紧凑预览信息

完整文件或选区范围、Region 数量、声道监听、响度匹配和当前操作状态现在都位于时间指标行。底部不再保留独立摘要和绿色状态区域；大波形会自动使用释放出来的高度。

窗口高度不足时，统一布局会从同一份实际可用高度中严格分配结果列表、分隔把手和预览区。大波形会逐级缩小最低高度，控制条不会再越出 PsyReaSFX 窗口。时间指标和 Pitch、Rate、Gain 已去除外框，只保留文字与参数轨道；交互图标继续使用统一边框。试听状态会省略已经在标题栏显示的重复文件名，并按剩余宽度安全裁切。

### 独立声道大波形

```text
设置 → 波形 → 独立显示各声道
```

- 单声道显示 `M`；
- 立体声分为 `L`、`R` 两条波形；
- 多声道分为 `CH 1` 至 `CH 8`；
- 选区、Region、时间标尺和播放指针贯穿全部声道，便于比较声道间的内容与同步关系。

独立声道使用新的 `RWF3` 高精度缓存。列表缩略图仍兼容既有 `RWF2` 缓存，不会因为升级而重建整个缩略图库。高精度预缓存会生成可直接复用的独立声道数据。

### 统一紧凑列表

结果列表使用单行垂直居中文字，不会在有限行高中强行绘制第二行分类文字。需要恢复默认字段时，可在表头右键选择“重置为默认字段”。

### 设置导航

左侧导航使用自绘双行卡片：第一行为页面名称，第二行为功能摘要。导航宽度和行高固定，不再依赖 `Selectable` 自动裁切，因此中文与英文标签都能完整显示。

### 色盘与即时预览

所有可修改颜色统一使用 ReaImGui 色盘：

```text
设置 → 外观 → 界面主题 / 波形配色
```

- 点击色块打开色盘；
- 选择颜色后立即应用到主界面；
- 色块旁显示当前 `#RRGGBB` 值，仅作为识别信息；
- 每个颜色项都有独立恢复按钮；
- 不再需要手动查找或输入颜色代码。

可调整项目包括强调色、普通/选中/已播放/已标记波形、已播放文字、选区、播放指针和 Region。


### 色块无法打开时

0.6.12 RC1 已移除覆盖色块的整行 `InvisibleButton`。颜色行现在使用非交互占位区域，点击事件会直接交给色块。

1. 点击颜色项右侧的小色块；
2. 色盘打开后直接选择颜色；
3. 颜色即时应用；
4. 点击“恢复”可还原该项默认值。

### About 页面

```text
设置 → 关于
```

页面显示：

- PsyReaSFX 版本与作者；
- REAPER、操作系统、ReaImGui 与 SWS 检测状态；
- 当前试听后端；
- 数据目录；
- 打开数据目录、打开文档目录；
- 复制诊断信息。

许可、官方网站与支持联系方式目前保留为待配置项。


## 5. 逻辑音效库与来源路径

左侧显示的是逻辑音效库，一个逻辑库可以拥有一个或多个实际文件夹。点击逻辑库会聚合显示全部来源路径；展开后可以只查看其中一条来源。悬停逻辑库可查看路径、在线状态和索引数量。

单击逻辑库名称前的箭头即可展开或折叠实际来源路径，不会改变当前筛选；展开状态会保存并在下次启动时恢复。

### 添加文件夹

- 点击 `+ 新建音效库`，先输入名称建立空逻辑库，不需要选择文件夹；来源路径可在之后逐步添加。
- 右键逻辑库并选择“添加来源路径…”，可继续挂载其他硬盘目录。
- 从资源管理器把文件夹拖到逻辑库、“全部音效库”或中央结果区。中央区域会沿用当前逻辑库上下文；没有库上下文时会建立新库。
- 一次拖入多个文件夹时，可以每个文件夹建立一个库，也可以合并为一个逻辑库。

完全重复的来源路径和相互包含的父子路径会被阻止，避免重复索引。把已经属于其他逻辑库的路径拖入目标库时，会先确认是否只移动其逻辑归属。以上操作都不会移动或删除硬盘文件。离线路径会继续显示，硬盘恢复后可重新扫描。

### 导入进度

新库导入会显示：

- 当前阶段和文件；
- 完成数、失败数和总数；
- 波形准备进度；
- 取消按钮。

### 管理库

`管理音效库` 支持重命名逻辑库、添加来源路径、扫描全部来源，以及单独打开、重建或移除某条来源。旧版根目录会自动迁移到 `libraries_v2.tsv`，成为“一库一路径”，不需要重建波形缓存。播放列表和项目素材箱仍是独立的虚拟集合。

## 7. 表头与字段

表头固定置顶。拖动列分隔线可修改列宽，双击恢复默认宽度。右键表头可显示或隐藏 Waveform、Filename、Status、Description、Category、Duration、Format、Library、Path 等字段。

Duration 列采用固定的 `MM:SS.mmm` 时间码，例如 `00:04.947`。

当字段总宽度超过中央工作区时，不显示常驻横向滚动条。鼠标悬停在固定表头或任意结果行，使用 `Shift + 鼠标滚轮` 横向查看；表头和行内容会同步移动，并可到达最右侧字段。用户设置的列宽会被保留，不会为了适应当前窗口而自动压扁全部字段。

隐藏字段不会参与行内绘制，有利于大型结果列表的性能。

### Artwork 列

在表头右键菜单勾选 `Artwork` 后，列表会显示方形封面缩略图。统一默认字段会启用该列。

Artwork 查找顺序：

1. 当前素材所在文件夹；
2. 逐级向上查找到音效库根目录，最多七层；
3. 优先识别 `artwork`、`cover`、`folder`、`front`、`album`、`thumbnail` 等 PNG/JPEG 文件；
4. 没有匹配名称时使用文件夹中第一张可用 PNG/JPEG；
5. 可在右侧元数据面板手动选择封面。

封面使用低优先级队列按需查找，不会在启动时扫描整库图片。图片内存缓存有数量上限，隐藏 Artwork 列时不会为不可见行加载缩略图。

PsyReaSFX 还会检查当前实体来源根目录及常见的 `Artwork`、`Images`、`Docs`、`Documentation` 子目录。每个实体来源独立拥有自己的封面：某个来源找到的图片不会扩散到同一逻辑库中的其他来源。展开逻辑库后，右键实体来源路径即可指定、重新自动查找或清除该来源的 Artwork；音效库管理器中每条来源旁的“封面”按钮也可手动选择。右侧元数据面板为单个素材指定的封面始终拥有最高优先级。


## 8. 选择、多选与标记

| 操作 | 功能 |
|---|---|
| 单击 | 单选 |
| Ctrl+单击 | 追加或取消单项 |
| Shift+单击 | 连续范围选择 |
| Ctrl+A | 全选当前结果 |
| F | 收藏或取消收藏 |
| M | 标记或取消标记 |

“收藏”和“标记”用途不同：收藏用于长期集合；标记用于当前筛选、视觉检查和临时处理状态。可使用 `marked:true` 搜索标记素材。

右键素材可批量收藏、标记、设置工作流状态、加入集合或分轨插入。

## 9. 状态颜色与本次已播放高亮

打开：

```text
设置 → 外观 → 颜色与状态
```

可设置：

- 普通、选中、已播放、已标记波形；
- 已播放文字颜色；
- 选区、播放指针和 Region 颜色。

### 已播放文字

素材在本次启动中真正开始试听后，Filename、Keywords/Description、Duration、Library 等文字列会改为可自定义的暖黄色。选中行仍保留蓝色背景，但已播放文字不会被蓝色选中状态覆盖。

这套颜色属于**当前启动会话**：

- 重新打开脚本后自动恢复普通文字颜色；
- 不会删除 `history_v1.tsv` 中的长期试听历史；
- `played:true` 搜索仍表示历史上试听过的素材；
- 主工具栏的“清除本次已播放高亮”图标可立即重置；
- 也可通过 `视图 → 清除本次已播放高亮` 或设置页执行。

可单独关闭“已播放文字高亮”。已播放波形高亮默认关闭，避免与标记、选中状态产生过多颜色竞争。

列表波形显示优先级仍为：

```text
选中 > 已标记 > 已播放波形（启用时） > 普通
```


## 10. 列表波形

列表波形默认 256 点，可切换为 512 点。点击小波形任意位置，会从对应时间开始试听。当前试听素材显示播放指针。

高精度整库预缓存支持 2048 或 4096 点，按素材逐个写入磁盘，不会把整库高精度波形长期留在内存。

## 11. 大波形

### 导航

- 鼠标滚轮：以鼠标位置为中心缩放；
- Shift+滚轮：水平平移；
- 中键拖动：水平平移；
- 双击：重置缩放；
- 右键拖动：擦播。

### 选区

左键拖动建立选区。可只试听选区、自动循环选区，或通过选区顶部拖拽区域直接拖入 REAPER。

### 时间与响度

顶部信息栏可显示素材名称、时长、声道、采样率、音效库以及可选的 LUFS-I、LUFS-M max、LUFS-S max、True Peak。响度结果按需分析并保存到 `loudness_v1.tsv`。

## 12. Region 与瞬态检测

一条长素材可保存多个非破坏性 Region。Region 数据保存在 `regions_v1.tsv`，不会修改源 WAV。

瞬态检测可设置：

- 阈值 dBFS；
- 包络平滑时间；
- 最小瞬态间隔；
- Region 前置和后置时间；
- 单次最大 Region 数；
- 是否替换已有建议。

自动检测 Region 标记为 `[T]`，手动 Region 标记为 `[M]`。支持撤销上次检测、清除全部瞬态建议或单独删除某个 Region。

## 13. 试听

支持：

- Pitch：-24 至 +24 半音；
- Rate：0.25x 至 4x；
- Gain：-36 dB 至 +18 dB；
- Preserve Pitch；
- Loop；
- Reverse；
- 原始、左、右和 Mono 监听；
- 估算响度匹配；
- Pitch 和 Rate 预设。

估算响度匹配用于快速比较，不等同于完整 LUFS 标准测量，只影响试听。

## 14. 插入与拖放

支持：

- 插入当前轨道；
- 插入新轨道；
- 按 BWF 时间戳插入；
- 只插入当前波形选区；
- 多选素材纵向分轨插入；
- 从列表或选区拖入 REAPER 编排区。

当前拖放是 PsyReaSFX 到 REAPER 的内部工作流，不是 Windows 通用文件拖放。

## 15. Transfer 导出

从底部工具条打开 Transfer，或按 `Ctrl+T`。Transfer 会生成新的音频文件，不修改源素材。

### 输出与命名

先选择输出目录，再设置命名模板。可用字段：

| 字段 | 内容 |
|---|---|
| `{name}` | 不含扩展名的源文件名 |
| `{category}` | Category 元数据 |
| `{subcategory}` | SubCategory 元数据 |
| `{library}` | 音效库名称 |
| `{index}` | 两位批次序号 |
| `{date}` | `YYYYMMDD` 格式的导出日期 |
| `{region}` | 当前保存 Region、`selection` 或 `full` |

生成的名称会自动处理 Windows 文件名非法字符；可在字段展开后统一转换为小写。

### 范围、格式与声道

- 当前素材：可导出大波形选区或完整文件；
- 多选素材：逐个导出每个完整源文件；
- 格式：WAV 24-bit PCM，或使用 REAPER 默认参数的 FLAC；
- 采样率：跟随源文件、44.1、48、96、192 kHz；
- 声道：跟随源声道、单声道、立体声。

### 处理与完成行为

导出会写入当前 Pitch、Rate、Gain、Reverse 和 Preserve Pitch。可增加渲染淡入淡出，并选择 Peak、True Peak 或 LUFS-I 标准化。重名策略支持自动递增、跳过，或经过明确确认后覆盖。完成的文件可选择自动插入 REAPER。

Transfer 会临时建立一个干声媒体对象，使用 REAPER 原生“已选媒体对象”渲染，然后恢复此前的渲染设置、对象与轨道选择、编辑光标、时间选区和工程修改状态。无论成功或失败，临时对象都会被移除。

Beta 1 不经过轨道 FX、发送、文件夹或 Master FX。运行 Transfer 前需要停止工程播放。Reverse Transfer 需要 SWS。

## 16. 集合与工作流状态

### 播放列表和项目素材箱

可以建立多个集合，批量加入或移除素材。删除集合不会删除文件。

### 保存搜索

保存搜索会记录关键词、库筛选、集合、工作流状态和排序条件。

### 工作流状态

素材可设置为：未标记、候选、已采用、已排除。该状态与 0.6.8 新增的独立 `marked` 标志不同。

## 17. 元数据与固定封面

右侧面板可非破坏性编辑 Description、Keywords、Category、SubCategory、CatID、Library 和 Artwork 路径。多选时必须先勾选需要批量写入的字段。

单选素材且启用 Artwork 时，右侧面板顶部会显示完整封面。默认开启“封面固定”：

- 封面、素材名和封面操作保持在面板顶部；
- 下方元数据与文件信息在独立区域中滚动；
- 滚动长元数据时封面位置不会随内容离开；
- 可手动选择封面、重新自动查找或明确清除封面；
- 关闭固定后，封面会与元数据一起滚动。

当前版本只修改 PsyReaSFX 数据库，不直接写回 WAV/BWF/iXML，也不提取音频文件内部嵌入式图片。Artwork 使用外部 PNG/JPEG 文件。


## 18. 界面与性能

### 界面

只维护一套紧凑、扁平、自适应界面，同时保留可调列宽、固定表头、左右面板折叠、专注模式和下方预览区高度调整。

### 当前性能策略

- 本次已播放状态使用路径哈希表，查询为常数时间；
- Artwork 使用文件夹负结果缓存，未找到图片的文件夹不会在每帧重复扫描；
- Artwork 查找为低优先级单任务队列，鼠标交互、扫描和导入时主动让步；
- 图片对象通过 ReaImGui Attach 持久化，并限制内存缓存数量；
- 只为当前可见 Artwork 行和选中素材请求图片；
- 紧凑行只绘制一行主文本，减少裁切和重复绘制；
- 设置中心仅在打开时绘制，不加入主浏览循环的数据库工作量。

### 通用性能原则

- 只绘制当前可见行；
- 隐藏字段不绘制；
- 扫描、元数据、响度和波形任务分帧执行；
- 鼠标交互时后台重任务让步；
- 高精度预缓存只写磁盘，不整库常驻内存；
- Region、集合、历史和响度使用独立轻量文件。

## 21. 维护与故障排查

设置中的维护页支持清空波形缓存、重置界面、保留库路径重建数据库和恢复出厂。

常见问题：

- `._文件名.wav` 显示波形不可用：它是 macOS AppleDouble 元数据旁车，不是音频。Beta 2 会在扫描时过滤，并在加载旧索引时自动忽略，不会删除磁盘文件。
- 波形首次显示较慢：等待可见行建立缓存，或执行高精度预缓存。
- 点击波形无法精确定位：安装或更新 SWS。
- 拖放失败：确认释放位置在 REAPER 编排区。
- 中文乱码：通常来自源文件元数据编码，而不是 PsyReaSFX 数据库。
- 大库卡顿：关闭不需要的列，使用专注模式，避免同时运行多个版本。
