# PsyReaSFX 更新日志（简体中文）

## 0.6.16 Stable RC5

### 阻断问题修复

- 修复 RC4 横向滚动固定表头引入的 ReaImGui `EndChild` 断言。
- 固定表头在结果 Child 之后调整交互坐标时，现在会提交一个零尺寸布局项目，满足 ReaImGui 对 `SetCursorScreenPos` 的边界要求。
- 修复不改变表头固定、横向滚动和分栏拖动功能。

## 0.6.15 Stable RC4

### 英文本地化

- 全面补齐设置、维护、外观、波形、帮助、状态栏和底部预览摘要的英文翻译。
- 为动态扫描、导入、预缓存、瞬态检测、集合与缓存迁移状态增加英文规则。
- 保持用户自己的文件名、音效库名称、元数据和集合名称原样显示。

### 结果列表

- 当可见字段总宽度超过工作区时显示横向滚动条。
- 固定表头与结果内容同步横向滚动。
- 保留用户设置的列宽，不再为了塞入当前窗口而强行压缩所有字段。

### 项目主页

- README 新增紧凑工作区主图和专注模式预览图。
- 脚本元数据与 About 页面固定保留 GitHub 项目主页。

## 0.6.14 Stable RC3

### 上次浏览高亮

- 新增 `last_played_session_v1.tsv`，独立保存上次浏览的黄色高亮快照。
- 新增“恢复上次浏览高亮”。
- 新增“启动时自动恢复上次浏览高亮”。
- 新增“清除已保存浏览记录”。
- “清除本次已播放高亮”只刷新当前界面，不立即删除可恢复快照。
- 自动保存和正常关闭时保存发生过变化的当前高亮集合。
- 完整试听历史继续独立保存在 `history_v1.tsv`。

### 色盘修复

- 修复点击色块没有反应。
- 移除覆盖整行的 `InvisibleButton`。
- 颜色行改用非交互 `Dummy` 预留布局。
- 点击事件现在直接到达 `ColorEdit3` 色块。
- 保留即时应用、色块预览、只读十六进制值和逐项恢复。

### 稳定候选

- 本版不新增大型模块，作为 0.6 Stable RC1。
- 通过 REAPER、ReaImGui 与 SWS 实机验证后，可晋升为最终 0.6 Stable。
- 继续集中会话状态、路径集合复制和计数逻辑，减少重复代码。

## 0.6.11

### 设置中心

- 修复左侧页面名称只显示单个字符或描述被裁切的问题。
- 导航改为自绘双行卡片，统一标题、摘要、选中条和悬停背景。
- 导航宽度增加，并新增 About 页面。

### 色盘

- 所有颜色字段改用 ReaImGui `ColorEdit3` 色盘入口。
- 点击色块即可打开完整色盘，选择后即时生效。
- 增加当前色块和 `#RRGGBB` 只读预览。
- 强调色与全部波形状态色统一复用同一组件。
- 移除重复的“应用颜色”步骤，保留逐项恢复和恢复默认配色。

### About

- 新增产品头部、版本、作者和 0.6 稳定化状态。
- 新增 REAPER、OS、ReaImGui、SWS 和试听后端信息。
- 新增打开数据目录、打开文档目录和复制诊断信息。
- 为许可、网站和支持联系方式预留正式发布字段。

### 代码与性能

- 集中颜色转换、色盘绘制和即时应用逻辑。
- 设置导航不再依赖多层文字覆盖绘制。
- 色盘仅在设置窗口打开时运行，不增加正常浏览时的后台任务。

## 0.6.10

### Forge 式已播放状态

- 已播放文字列改为当前启动会话状态，默认暖黄色并可自定义。
- Filename、Keywords/Description、Duration、Library 等文字列统一响应已播放颜色。
- 重新打开脚本时自动恢复普通颜色，不删除长期试听历史。
- 主工具栏新增“清除本次已播放高亮”图标。
- `视图`菜单和设置页提供同一手动重置命令。
- 已播放波形高亮改为独立可选项，默认关闭。

### 紧凑结果列表

- 新增 Forge 紧凑完整预设。
- 紧凑行改为单行垂直居中，修复第二行分类文字被遮挡。
- 预设默认显示 Waveform、Filename、Keywords/Description、Artwork、Duration。
- 可从设置预设卡或表头右键菜单应用。

### Artwork

- 新增可选 Artwork 列和列表封面缩略图。
- 支持 PNG/JPEG 文件夹封面自动发现。
- 支持向上查找到音效库根目录并设置最大搜索层级。
- 右侧元数据面板新增完整封面、手动选择、重新检测和清除。
- 新增封面固定选项；元数据滚动时封面保持在面板顶部。
- Artwork 路径保存到 PsyReaSFX 数据库，不修改音频文件。

### 设置中心

- 设置窗口改为现代双栏布局。
- 新增 Forge 紧凑、Aether 标准和检查/审阅三张预设卡。
- 外观、Artwork、状态颜色、波形和维护选项重新分组。
- 设置窗口风格与主界面统一。

### 代码与性能

- 作者字段改为 `Psysia`。
- 新增 Artwork 文件夹正/负结果缓存，避免无封面目录反复扫描。
- Artwork 使用低优先级单任务队列，并在用户交互、扫描和导入时让步。
- 图片对象限制内存缓存数量，清理时主动 Detach。
- 只为可见 Artwork 行和选中素材加载图片。
- 状态颜色查询统一使用路径哈希表。
- 继续保持版本在 0.6.x 稳定化阶段。


## 0.6.9

### 修复

- 修复下半部分图标悬停说明不显示。
- 图标说明改用 ReaImGui 前景绘制层，确保显示在所有 Child 面板之上。
- 顶部和底部图标继续共用同一套延迟与防闪烁逻辑。

### 外观系统重构

- 删除差异不明显的“清爽卡片 / 紧凑模式”。
- 新增舒适、标准、紧凑三种全局界面密度。
- 密度会实际改变列表行高、表头高度、图标、参数卡片、间距和圆角。
- 新增扁平、分层、高对比三种层级风格。
- 新增完整、专注、极简三种预览控制台。
- 专注和极简模式将低频试听开关移入“更多操作”，功能不会丢失。
- 自动迁移旧版 `ui_style` 和控制台布局配置。

### 性能

- 外观变化只修改尺寸、颜色和布局参数。
- 不增加波形、响度、扫描或数据库后台任务。

## 0.6.8

### 补齐 0.6 计划项

- 完成普通、选中、已播放、已标记四种列表波形状态配色。
- 增加可配置的 `普通波形`、`选中波形`、`已播放波形`、`已标记波形` 颜色。
- 固定显示优先级：选中 > 已标记 > 已播放 > 普通。
- 已标记波形增加小圆点提示。

### 标记系统

- 新增独立 `marked` 素材状态。
- 新增 `M` 快捷键。
- 新增单项和批量标记/取消标记右键操作。
- 新增 `marked:true`、`marked:false` 搜索。
- 新增 `played:true`、`played:false` 搜索。
- `marked` 字段写入 `index_v3.tsv`，旧数据库自动兼容。

### 代码整理与性能

- 集中建立波形状态解析函数，移除行绘制中的重复颜色分支。
- 单行绘制只计算一次规范化路径键。
- 波形配色字段改为数据驱动定义，减少设置界面的重复代码。
- 增加波形配色一键恢复默认值。
- 本次功能不增加波形分析、磁盘扫描或响度计算任务。

### 文档

- README 改为发布入口页。
- 用户说明书与更新日志拆分为独立文档。
- 中文和英文分别维护，共四份正式文档。
